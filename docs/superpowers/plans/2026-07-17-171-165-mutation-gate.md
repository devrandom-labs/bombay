# Reproducible mutation gate (#171 + #165) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A standing, fully-reproducible mutation gate that fails on survivors, timeouts, incomplete runs, or per-function viability collapse — and always reports the "N viable / M total" ratio so "0 survivors" stops being a PR-body assertion.

**Architecture:** A pure Rust tool (`mutants-gate`) reads cargo-mutants' `outcomes.json` + `mutants.json` against a committed `mutants-baseline.json` and owns the verdict. The flake's `packages.mutants` derivation runs the sweep then the gate (`nix build .#mutants`); a nightly workflow triggers the flake. Ratchet is keyed per `file::function`.

**Tech Stack:** Rust (edition 2024, stable), serde/serde_json, cargo-mutants 27.0.0, Nix/crane, GitHub Actions.

## Established facts (verified against cargo-mutants 27.0.0 in the devshell during planning — do NOT re-verify)

- `outcomes.json` shape: `{ "total_mutants": u64, "caught": u64, "missed": u64, "timeout": u64, "unviable": u64, "success": u64, "outcomes": [ { "summary": <Summary>, "scenario": <Scenario>, ... } ], ... }`.
- `Summary` is an externally-tagged bare-string enum: `"Success"` (baseline only), `"CaughtMutant"`, `"MissedMutant"`, `"Unviable"`, `"Timeout"`.
- `scenario` is either the bare string `"Baseline"` **or** `{ "Mutant": { "file": "<path>", "function": { "function_name": "<name>", ... }, ... } }`.
- `mutants.json` is a **top-level array** of candidates; each has `{ "file": "<path>", "function": { "function_name": "<name>" }, ... }`. Its length is the authoritative candidate count for the completeness check.
- Scoping `--package bombay-core --package bombay_macros --file 'bombay-core/**' --file 'macros/src/derive_msg.rs'` mutates exactly the bombay-core surface + `macros/src/derive_msg.rs`, **excluding** vendored kameo derives. Confirmed via `cargo mutants --list`.
- Macros package name is `bombay_macros` (underscore); `bombay-core` (hyphen).
- A clean scoped `kind.rs` sweep reproduces the card baseline exactly: 8 mutants, 2 caught, 6 unviable.

```rust
//! The pure heart of the gate: turn a parsed run + committed baseline into a
//! verdict. No IO here — `main` reads the files and calls `evaluate`.

use std::collections::BTreeMap;

use crate::model::{Baseline, Candidate, Report, Scenario, Summary};

/// Per-`file::function` mutation tally.
#[derive(Debug, Default, Clone, PartialEq, Eq)]
pub(crate) struct Tally {
    pub(crate) total: u64,
    pub(crate) caught: u64,
    pub(crate) unviable: u64,
    pub(crate) missed: u64,
    pub(crate) timeout: u64,
}

impl Tally {
    /// A mutant is "viable" when it compiled and ran (caught, missed, or
    /// timed out) — i.e. everything except `Unviable`. This is the number the
    /// ratchet floors, because unviable mutants measure nothing about tests.
    pub(crate) fn viable(&self) -> u64 {
        self.caught.saturating_add(self.missed).saturating_add(self.timeout)
    }
}

/// A reason the gate fails. One variant per failure domain.
#[derive(Debug, PartialEq, Eq)]
pub(crate) enum Failure {
    /// Fewer mutant outcomes than candidates — the run was interrupted (#148).
    Incomplete { candidates: u64, executed: u64 },
    /// A mutant survived (a test SHOULD have caught it and did not).
    Survivor { key: String },
    /// A mutant timed out — treated as a failure, per cargo-mutants + #148.
    Timeout { key: String },
    /// A floored function's viable count dropped below its recorded floor.
    Collapse { key: String, floor: u64, viable: u64 },
    /// A function the sweep saw is in neither `floors` nor `known_zero_viable`.
    Unaccounted { key: String, viable: u64 },
}

/// The full outcome of a gate run: the tally (always) + any failures.
#[derive(Debug)]
pub(crate) struct Verdict {
    pub(crate) tallies: BTreeMap<String, Tally>,
    pub(crate) failures: Vec<Failure>,
}

impl Verdict {
    pub(crate) fn passed(&self) -> bool {
        self.failures.is_empty()
    }

    pub(crate) fn total_viable(&self) -> u64 {
        self.tallies.values().map(Tally::viable).sum()
    }

    pub(crate) fn total_mutants(&self) -> u64 {
        self.tallies.values().map(|t| t.total).sum()
    }
}

fn key(file: &str, function: &str) -> String {
    format!("{file}::{function}")
}

/// Group mutant outcomes (skipping the `Baseline`) into per-function tallies.
fn tally(report: &Report) -> BTreeMap<String, Tally> {
    let mut by_fn: BTreeMap<String, Tally> = BTreeMap::new();
    for outcome in &report.outcomes {
        let Scenario::Mutant(info) = &outcome.scenario else {
            continue;
        };
        let entry = by_fn.entry(key(&info.file, &info.function.function_name)).or_default();
        entry.total = entry.total.saturating_add(1);
        match outcome.summary {
            Summary::CaughtMutant => entry.caught = entry.caught.saturating_add(1),
            Summary::Unviable => entry.unviable = entry.unviable.saturating_add(1),
            Summary::MissedMutant => entry.missed = entry.missed.saturating_add(1),
            Summary::Timeout => entry.timeout = entry.timeout.saturating_add(1),
            Summary::Success => {}
        }
    }
    by_fn
}

/// Evaluate a run against the baseline. Pure: no IO, deterministic.
pub(crate) fn evaluate(report: &Report, candidates: &[Candidate], baseline: &Baseline) -> Verdict {
    let tallies = tally(report);
    let mut failures = Vec::new();

    // 1. Completeness — did every candidate produce an outcome?
    let executed: u64 = tallies.values().map(|t| t.total).sum();
    let candidate_count = candidates.len() as u64;
    if executed < candidate_count {
        failures.push(Failure::Incomplete { candidates: candidate_count, executed });
    }

    let known_zero: std::collections::BTreeSet<&str> =
        baseline.known_zero_viable.iter().map(String::as_str).collect();

    for (k, t) in &tallies {
        // 2 & 3. Survivors and timeouts fail regardless of the floor.
        if t.missed > 0 {
            failures.push(Failure::Survivor { key: k.clone() });
        }
        if t.timeout > 0 {
            failures.push(Failure::Timeout { key: k.clone() });
        }

        // 4 & 5. Ratchet + accounting.
        let viable = t.viable();
        match baseline.floors.get(k) {
            Some(&floor) => {
                if viable < floor {
                    failures.push(Failure::Collapse { key: k.clone(), floor, viable });
                }
            }
            None => {
                if !known_zero.contains(k.as_str()) {
                    // Not floored and not documented-zero: the baseline does not
                    // account for this function. Force a conscious classification.
                    failures.push(Failure::Unaccounted { key: k.clone(), viable });
                }
            }
        }
    }

    Verdict { tallies, failures }
}
```

```rust
/// Render the always-printed ratio + per-function table.
pub(crate) fn render_report(v: &Verdict) -> String {
    use std::fmt::Write as _;
    let mut s = String::new();
    let _ = writeln!(s, "mutation coverage: {} viable / {} total", v.total_viable(), v.total_mutants());
    let _ = writeln!(s, "{:<60} {:>6} {:>6} {:>8}", "file::function", "viable", "total", "unviable");
    for (k, t) in &v.tallies {
        let _ = writeln!(s, "{:<60} {:>6} {:>6} {:>8}", k, t.viable(), t.total, t.unviable);
    }
    s
}

/// Build a baseline skeleton from a clean run: every viable function floored at
/// its current viable count; every 0-viable function listed as known-zero.
/// The operator REVIEWS the diff before committing (a genuinely-should-be-tested
/// 0-viable function is caught here, not by the machine).
pub(crate) fn emit_baseline(report: &Report) -> String {
    let tallies = tally(report);
    let mut floors = std::collections::BTreeMap::new();
    let mut known_zero = Vec::new();
    for (k, t) in &tallies {
        if t.viable() > 0 {
            floors.insert(k.clone(), t.viable());
        } else {
            known_zero.push(k.clone());
        }
    }
    let value = serde_json::json!({ "floors": floors, "known_zero_viable": known_zero });
    serde_json::to_string_pretty(&value).unwrap_or_else(|_| "{}".to_owned())
}
```

```rust
//! CI tool: turns a cargo-mutants run into a pass/fail verdict that cannot be
//! vacuously green. See docs/adr/0006-mutation-viable-ratchet.md.
//!
//! Usage:
//!   mutants-gate check <mutants.out-dir> <baseline.json>
//!   mutants-gate emit-baseline <mutants.out-dir>

mod gate;
mod model;

use std::path::{Path, PathBuf};
use std::process::ExitCode;

use model::{Baseline, Candidate, Report};

#[derive(Debug, thiserror::Error)]
enum GateError {
    #[error("usage: mutants-gate <check <out-dir> <baseline.json> | emit-baseline <out-dir>>")]
    Usage,
    #[error("reading {path}: {source}")]
    Io { path: PathBuf, #[source] source: std::io::Error },
    #[error("parsing {path}: {source}")]
    Parse { path: PathBuf, #[source] source: serde_json::Error },
}

fn read_json<T: serde::de::DeserializeOwned>(path: &Path) -> Result<T, GateError> {
    let raw = std::fs::read_to_string(path).map_err(|source| GateError::Io { path: path.to_owned(), source })?;
    serde_json::from_str(&raw).map_err(|source| GateError::Parse { path: path.to_owned(), source })
}

fn run() -> Result<bool, GateError> {
    let mut args = std::env::args().skip(1);
    let Some(mode) = args.next() else { return Err(GateError::Usage) };
    match mode.as_str() {
        "emit-baseline" => {
            let Some(dir) = args.next() else { return Err(GateError::Usage) };
            let report: Report = read_json(&Path::new(&dir).join("outcomes.json"))?;
            println!("{}", gate::emit_baseline(&report));
            Ok(true)
        }
        "check" => {
            let (Some(dir), Some(baseline_path)) = (args.next(), args.next()) else {
                return Err(GateError::Usage);
            };
            let dir = Path::new(&dir);
            let report: Report = read_json(&dir.join("outcomes.json"))?;
            let candidates: Vec<Candidate> = read_json(&dir.join("mutants.json"))?;
            let baseline: Baseline = read_json(Path::new(&baseline_path))?;
            let verdict = gate::evaluate(&report, &candidates, &baseline);
            print!("{}", gate::render_report(&verdict));
            for failure in &verdict.failures {
                eprintln!("mutants-gate FAIL: {failure:?}");
            }
            Ok(verdict.passed())
        }
        _ => Err(GateError::Usage),
    }
}

fn main() -> ExitCode {
    match run() {
        Ok(true) => ExitCode::SUCCESS,
        Ok(false) => ExitCode::FAILURE,
        Err(err) => {
            eprintln!("mutants-gate: {err}");
            ExitCode::from(2)
        }
    }
}
```

name: mutants

      - main

        # Always run so a FAILED gate still surfaces the ratio (that is the

              echo '### Mutation coverage'

              echo '```'

          for result in ${{ join(needs.*.result, ' ') }}; do
            if [ "$result" != "success" ]; then
              echo "::error::the mutants sweep finished with result: $result"
              exit 1
            fi
          done
